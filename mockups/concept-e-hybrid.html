From concept B: the preview sits beside the form at all times so beginners get instant feedback without hunting for a "preview" button.
Batch boxes: each box is one build plate sent to a printer, with an editable name. Rows reuse the queue's compact style. The checkmark confirms a row printed; clicking it opens a tiny inline amount input (default the full count) so a partial print can be confirmed. The X marks the row failed and returns the row or remaining amount to the main queue. Confirmed rows vanish from the batch; a batch box that empties disappears entirely. "Download" is a split button with three formats; there is deliberately no plate preview.
