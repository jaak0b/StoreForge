Design decision. The shorthand omnibox is the single entry point for everything: add a bin, run a screw import, or jump the plate composer with a slash command. No separate "add" dialog, no import wizard screen: one input, always in the same place, always focused on load.

Parse happens on every keystroke, no submit-then-fail round trip. Bad shorthand shows the failing token inline in red with a plain-English reason ("no thread size found for '3x20fhc'") rather than a toast that vanishes.
