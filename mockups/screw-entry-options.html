Screw entry tab: replacement options

The owner rejected the structured row table (concept E) because it acted as a second, staged queue in front of the real one ("Add 3 bins to queue"). Every option below removes that staging step entirely: whatever you type or pick lands in the real queue immediately, with a subtle highlight on the row that was just added.

Fastest path for anyone who already knows the shorthand: type, watch it parse live into real bin previews, hit Enter once and everything is queued, no confirmation screen.

OPTION 2

Single-screw picker, mirrors Manual bin

For anyone who doesn't know the shorthand: the exact same form language as the Manual bin tab (four fields, preview beside it, one Add button), so the two tabs feel like the same app instead of two different tools.

The form keeps its values after adding, so changing just one field (say, length) and hitting Add again is the fast path for a screw box with several lengths of the same thread.
